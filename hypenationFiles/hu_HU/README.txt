The Free Hungarian Dictionary is a free-to-use Hungarian dictionary created for the Hunspell spelling checker and morphological analysis program. It provides uniquely accurate spell checking for users of the free LibreOffice office suite, Mozilla Firefox browser and Thunderbird mail program, Google Chrome, Internet Explorer (as an add-on), Opera, SDL Trados, MemoQ and macOS (installer).
